Load data set into sfaira data format.

Parameters

  • data – Data set.

  • organism – Organism of data set.

  • organ – Organ of data set.

  • gene_symbol_col – Column name in .var which contains gene symbols. Set to “index” to use the index.

  • gene_ens_col – Column name in .var which contains ENSG symbols. Set to “index” to use the index.

  • obs_key_celltypes – .obs column name which contains cell type labels.

  • class_maps – Cell type class maps.

  • dataset_id – Identifer of data set.
