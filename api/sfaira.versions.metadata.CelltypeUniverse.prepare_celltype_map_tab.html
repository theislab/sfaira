sfaira.versions.metadata.CelltypeUniverse.prepare_celltype_map_tab¶

CelltypeUniverse.prepare_celltype_map_tab(source, match_only: bool = False, include_synonyms: bool = True, anatomical_constraint: Optional[str] = None, omit_list: list = [], n_suggest: int = 10, separator_suggestions: str = ':', separator_groups: str = ':|||:') → pandas.core.frame.DataFrame¶

Map free text node names to ontology node names via fuzzy string matching and return as matching table.

Parameters

  • source – Free text node labels which are to be matched to ontology nodes.

  • match_only – Whether to include strict matches only in output.

  • include_synonyms – Whether to include synonyms of nodes in string search.

  • anatomical_constraint – Whether to require suggestions to be within a target anatomy defined within UBERON.

  • omit_list – Free text node labels to omit in map.

  • n_suggest – Number of cell types to suggest per search strategy.

  • separator_suggestions – String separator for matches of a single strategy in output target column.

  • separator_groups – String separator for search strategy grouped matches in output target column.

Returns

Table with source and target node names. Columns: “source”, “target”
